# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-22

### Added
- Initial release of `@oyerinde/caliper-bridge`.
- WebSocket integration with `@oyerinde/caliper-mcp`.
- High-precision intent handling for `CALIPER_SELECT`, `CALIPER_MEASURE`, and `CALIPER_INSPECT`.
- Passive state exporter with automatic element tagging (Caliper IDs).
- Security gate via `data-caliper-enable-agent` attribute.
