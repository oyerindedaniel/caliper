# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-24

### Added
- Initial release of `@oyerinde/caliper-mcp`.
- **MCP tools**: `caliper_list_tabs`, `caliper_switch_tab`, `caliper_inspect`, `caliper_measure`, `caliper_get_state`, `caliper_clear`, `caliper_audit_node`, `caliper_grep`, `caliper_walk_dom`, and `caliper_parse_selector`.
- **MCP prompts**: `caliper-spacing-check`, `caliper-figma-audit`, and `caliper-audit-harness`.
- **MCP resources**: `caliper://state` and `caliper://tabs`.
- CLI support for dynamic port assignment via `--port/-p`.
- Zero-dependency runtime.
